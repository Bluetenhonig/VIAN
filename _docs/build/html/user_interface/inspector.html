Inspector¶

The Inspector is one of the most important Widgets of VIAN. The Inspector is content-sensitive, meaning that it adapts to the current selection. Most of the modifications to a specific item (i.e. a Segmentation, Segment, Annotation, Screenshot, etc.) can be performed using the Inspector.

The Inspector consists of four distinct parts, which are separated by black lines:

  • Title and Name
  • Item specific Attributes
  • Vocabulary Section
  • Notes Section

The Inspector showing a Screenshot. Note the four distinct Sections.
