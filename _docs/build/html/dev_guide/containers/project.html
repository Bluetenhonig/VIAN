class core.container.project.VIANProject(main_window, path='', name='', folder='')[source]¶

This Class hold the Complete VIAN Project. As such it is the owner of all subsequent IProjectContainers, directly or indirectly

A VIANProject is a FileSystem and a json File that is located inside this folder.

:var undo_manager = UndoRedoManager() :var main_window: A Reference to the Main Window :var inhibit_dispatch: if Dispatch should be inhibited

Variables:
  • path – The Path to the Projects .eext file (absolute)
  • name – The Name of the Project
  • folder – The Root Directory of the Project where the eext file lies inside
  • data_dir – the path to the data dir
  • results_dir – the path to the result dir
  • shots_dir – the path to the shot dir
  • export_dir – the path to the export dir
  • corpus_id – this is the VIANCorpus id of this project (default: -1)
  • id_list – A list of [Unique-ID, IProjectContainer] tuples
  • annotation_layers – A List of Annotation Layers
  • current_annotation_layer – The Currently Selected Annotation Layer
  • screenshots – A List of all Screenshots
  • segmentation – A List of all Segmentations
  • main_segmentation_index – The index of the main Segmentation Layer in self.segmentations
  • movie_descriptor – The Movie Descriptor of this Project
  • analysis – A List of IAnalysisResult
  • screenshot_groups – All Screenshot Groups
  • vocabularies – All Vocabularies (deprecated these are now global)
  • experiments – A List of all Experiments
  • current_script – The Currently Selected NodeScript
  • node_scripts – A List of All Node Scripts

  • colormetry_analysis – The ColorimetryAnalysis Reference
  • selected – A List of Currently Selected IProjectContainers
