Segmentation Module¶

class core.container.segmentation.Segment(ID=None, start=0, end=1000, duration=None, segmentation=None, annotation_body='', name='New Segment')[source]¶
Variables

  • MIN_SIZE – The Shortest size in MS

  • ID – The Segments ID in the Segmentation {0, …, n}

  • start – Time start in MS

  • end – Time end in MS

  • name – the Name of the Segment

  • duration – The Duration of the Segment in MS

  • annotation_body – The Annotation Content Text

  • timeline_visibility – If it is visible in the Timeline

  • segmentation – it’s parent Timeline

  • notes – Additional Notes that can be set in the Inspector

class core.container.segmentation.Segmentation(name=None, segments=None)[source]¶
Variables

  • name – The Name of the Segmentation

  • segments – A List of Segments

  • timeline_visibility – if it is visible in the timeline or not

  • notes – Additional Notes that can be added to describe it in the Inspector

  • is_main_segmentation – If this is the main Segmentation
