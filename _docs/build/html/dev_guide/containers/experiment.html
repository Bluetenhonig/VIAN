Experiment Module¶

class core.container.experiment.ClassificationObject(name, experiment, parent=None)[source]¶

A ClassificationTarget is an Object that one wants to classify by a set of Vocabularies. Several ClassificationTargets may form a Tree.

Example: Say one wants to analyse the Foreground and Background Color of a given Film using his homemade Vocabulary called “ColorVocabulary”.

The ClassificationTargets would therefore be “Foreground” and “Background”, both will have “ColorVocabulary”.

Variables:
  • name – The Name of this ClassificationObject
  • experiment – A reference to the Experiment it belongs to
  • parent – A Parent Classification Object or an Experiment if it’s at the root
  • children – A List of Chilren ClassificationObjects if any
  • classification_vocabularies – A List of Vocabularies attached to thsi ClassificationObject
  • unique_keywords – A List of Unique Keywords generated from this ClassificationObjects and its Vocabularies
  • target_container – A List of Target Containers to classify with this Classification Object
class core.container.experiment.Experiment(name='New Experiment')[source]¶

An Experiment holds all information connected to Classification of Objects. As such it defines rules for an experiment and tracks the Progress.

Variables:
  • name – The Name of this Experiment
  • classification_objects – The Classification Objects Attached to it
  • analyses – The Names of Analyses that have to be performed in this experiment
  • analyses_parameters – The List of Analyses parameters connected to the analyses above
  • classification_results – The Classification Mapping a list of [IClassifiable, UniqueKeywords]

class core.container.experiment.UniqueKeyword(experiment, voc_obj: core.container.experiment.Vocabulary = None, word_obj: core.container.experiment.VocabularyWord = None, class_obj: core.container.experiment.ClassificationObject = None)[source]¶

Unique Keywords are generated when a Vocabulary is added to a Classification Object. For each word in the Vocabulary a Unique Keyword is created to the Classification Object.

Variables:
  • experiment – The Experiment this Keyword belongs to
  • voc_obj – The Vocabulary this keyword origins
  • word_obj – The VocabularyWord this keyword origins
  • class_obj – The ClassObj this keyword origins
  • external_id – An External Key for the ERC-FilmColors Project
