Annotations Module¶

class core.container.annotation.Annotation(a_type=None, size=None, color=(255, 255, 255), orig_position=(50, 50), t_start=0, t_end=-1, name='New Annotation', text='', line_w=2, font_size=10, resource_path='', tracking='Static')[source]¶
Variables

  • name – name

  • a_type – An AnnotationType Enum value: { Rectangle = 0, Ellipse = 1, Line = 2, Text = 3, Image = 4, FreeHand = 5 }

  • t_start – The Start Time in MS

  • size – The Rect Size In original space

  • curr_size – The Rect Size in relative space to the currently displayed movie frame

  • color – The Color of the Annotation

  • orig_position – The Position in original space

  • line_w – The Line thickness

  • resource_path – Ressource Path if there should be one (Image Annotation)

  • text – Text if any (TextAnnotation)

  • font_size – Font Size if any (Text Annotation)

  • font – FontFamily Name if any (Text Anntotation)

  • has_key – If it is Keyed or not

  • keys – A List of (Time, Position) Tuples

  • free_hand_paths – A List of drawing Paths in form [path, color, width]

  • notes – Additional notes set in the Inspector

  • is_automated – If this Annotation content is driven by another variable

  • automated_source – The Source object hat is used in driving this ones content

  • automate_property – The Source object’s property that is driving this ones content

  • tracking – tracking

  • annotation_layer – A Reference to it’s parent Annotation Layer

  • is_visible – If this is globaly visible or not

  • widget – A Reference to it’s widget in the DrawingOverlay

  • image – An Image data if there is any (Image Annotation)

get_color()[source]¶

Returns the annotations color as QColor :return: QColor(self.color)

class core.container.annotation.AnnotationLayer(name=None, t_start=0, t_end=0)[source]¶

Member Variables: :var name: The Name of this Annotation Layer :var t_start: Start Time in MS :var t_end: End Time in MS :var annotations: A List of Annotations :var is_current_layer: If this is the current layer to edit :var is_visible: if this layer is currently visible or hidden :var timeline_visibility: If the layer should be shown in the Timeline or not :var notes: Additional notes set in the Inspector
